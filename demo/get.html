getRealMonth()

関数概要

UltraDate.prototype.getRealMonth()

  • 「Date.getMonth()」では実際の日付から1引かれた数字が返されますが、 「getRealMonth()」では実際の月の数字が返されます
  • 引数：なし
  • 戻り値：Number

戻り値

型 説明
Number 実際の月の数字

サンプル

実際の月の数字の取得

getEndDate()

関数概要

UltraDate.prototype.getEndDate(num)

  • 引数「num」でオフセットした月の月末の日付を取得します
  • 引数：(Number num)
  • 戻り値：Number

引数

名前 型 説明
num Number
  • オフセットしたい月数の数字を指定します
  • 正数の時は月を増やす
  • 負数の時は月を減らす

戻り値

型 説明
Number 月末の日付の数字

サンプル

月末の日付を取得

getHolidays()

関数概要

UltraDate.prototype.getHolidays(year, locale)

  • 引数「year」で指定された年、引数「locale」で指定されたロケールの祝祭日一覧のオブジェクトを取得します
  • 引数：(Number year, String locale)
  • 戻り値：Object

引数

名前 型 説明
year Number
  • 取得する年を数字で指定します
  • 指定の無い場合は現在のDateオブジェクトの年が使用されます
locale String
  • 取得するロケールの文字列を指定します
  • 指定の無い場合はデフォルトのロケールが使用されます

戻り値

型 説明
Object 祝祭日一覧のオブジェクトを返します

サンプル

日本の祝祭日の一覧を取得

  • 「UltraDate.ja.js」を読み込んでいるものとする

UltraDate.prototype.getHoliday(locale)
