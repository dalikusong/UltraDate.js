format()

関数概要

UltraDate.format(format, useWrap, eraStrict, locale)

  • 引数「format」で指定されたフォーマットで文字列を返します
  • 引数：(String format, Boolean useWrap, Boolean eraStrict, String locale)
  • 戻り値：String

引数

名前 型 説明
format String
  • 変換する文字列を指定します
  • フォーマット文字列については下記を参照してください
useWrap Boolean
  • 引数「format」の中にフォーマット用文字列が含まれる場合に使用します
  • フォーマット用文字列は前後に「_」を付与した文字列になります（例：「yyyy」→「_yyyy_」）
  • 「true」が指定された時のみ適用されます
eraStrict Boolean
  • 厳格な元号計算「eraStrict」を使用するかどうかを指定します
  • 「true」が指定された時のみ適用されます
locale String
  • 省略された場合はデフォルトのフォーマットが使用されます
  • 指定されたロケールが存在しない場合は「def」が使用されます

戻り値

型 説明
String 変換された文字列

文字列 変換内容

eeee 元号計算（フォーマットの設定の「era[year]」）で出された年の右4桁を出力。桁数が足りない場合は左0埋め（0000～9999）
eee 元号計算（フォーマットの設定の「era[year]」）で出された年の右3桁を出力。桁数が足りない場合は左0埋め（000～999）
ee 元号計算（フォーマットの設定の「era[year]」）で出された年の右2桁を出力。桁数が足りない場合は左0埋め（00～99）

gg 短い形式の元号（フォーマットの設定の「era[shortName]」）
g アルファベットの頭文字（フォーマットの設定の「era[alphaName]」）
