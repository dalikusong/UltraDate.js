setRealMonth()

関数概要

UltraDate.prototype.setRealMonth(num)

  • Dateオブジェクトでは月の設定は実際の月から1を引いた値をセットしますが、この関数では実際の月でセットできます
  • チェーンメソッドに対応しています
  • 引数：(Number num)
  • 戻り値：なし(return this;)

引数

名前 型 説明
num Number
  • セットしたい実際の月の数字を指定します
  • 正数の時はミリ秒を増やす
  • 負数の時はミリ秒を減らす

サンプル

実際の月で月をセット

setDayCountsInMonth()

関数概要

UltraDate.prototype.setDayCountsInMonth(count, day)

  • 第3土曜日みたいに第[count][day]曜日に日付をセットします
  • チェーンメソッドに対応しています
  • 引数：(Number count, Number day)
  • 戻り値：なし(return this;)

引数

名前 型 説明
count Number
  • 何回目かを指定します
  • 必須項目です
  • 数字にパースできない場合はErrorをスローします
day Number
  • 曜日を表す 0 (日曜日) ～ 6 (土曜日) の整数を指定します
  • 必須項目です
  • 0～6以外の時や数字にパースできない場合はErrorをスローします

サンプル

第[count][day]曜日をセット

setBeforeWeekday()

関数概要

UltraDate.prototype.setBeforeWeekday(option, locale)

  • 日付を引数「option」でしたものの前の平日に変更
  • チェーンメソッドに対応しています
  • 引数：(Number option, String locale)
  • 戻り値：なし(return this;)

引数

名前 型 説明
option Number
  • 0：土曜日、日曜日、引数「locale」で取得される祝祭日一覧
  • 1：土曜日、日曜日のみ
  • 2：日曜日、引数「locale」で取得される祝祭日一覧のみ
  • 3：日曜日のみ
  • 4：引数「locale」で取得される祝祭日一覧のみ
locale String
  • 取得するロケールの文字列を指定します
  • 指定の無い場合はデフォルトのロケールが使用されます
